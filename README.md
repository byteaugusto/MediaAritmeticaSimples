# MediaAritmeticaSimplesUm projeto simples em C# com Windows Forms que calcula a média aritmética de três valores inseridos pelo usuário. O programa valida os valores para garantir que estão entre 0 e 10, destacando campos inválidos em vermelho.

Ideal para iniciantes em C# e Windows Forms, serve como exemplo de validação de entrada, interação com o usuário e lógica de cálculo.

🛠 Funcionalidades
Entrada de três valores numéricos entre 0 e 10.
Validação de dados: valores inválidos são destacados em vermelho.
Cálculo automático da média quando todos os campos estão válidos.
Exibe a média com duas casas decimais.
📌 Como usar
Abra o projeto no Visual Studio.
Compile e execute.
Insira valores de 0 a 10 nos três campos.
Clique em Calcular Média.
O resultado aparecerá na label abaixo dos campos.

⚠️ Valores inválidos (fora do intervalo 0–10) ficarão com o fundo vermelho.

💻 Tecnologias utilizadas
Linguagem: C#
Plataforma: .NET Framework / Windows Forms
Ambiente de desenvolvimento: Visual Studio
